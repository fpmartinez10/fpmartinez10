### Selected Work

**[mesa](https://github.com/cruk-mi/mesa)** — R package for Methylation Enrichment Sequencing Analysis.
Provides tidy, Bioconductor-compatible tools for MBD-seq QC, normalisation, and differential methylation.
Developed at CRUK National Biomarker Centre.
`R` `Bioconductor` `MBD-seq` `epigenomics`

---

**[methylation-pipeline](https://github.com/fpmartinez10/methylation-pipeline)** — Nextflow DSL2 pipeline for
whole-genome bisulfite sequencing: trimming, alignment (Bismark), CpG extraction, and DMR calling.
Configurable for SLURM HPC and AWS Batch with Singularity containers.
`Nextflow` `nf-core` `WGBS` `Bismark` `AWS`

---

**[hpc-nextflow-templates](https://github.com/fpmartinez10/hpc-nextflow-templates)** — Production-ready
Nextflow DSL2 skeleton workflows for SLURM HPC environments: executor config, per-process resource profiles,
Singularity execution, and GitHub Actions CI.
`Nextflow` `SLURM` `Singularity` `GitHub Actions`
